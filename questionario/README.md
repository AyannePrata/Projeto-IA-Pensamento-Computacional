# Questionário de Avaliação

Esta pasta contém o instrumento de coleta de dados utilizado na pesquisa.

## Conteúdo

- **Questionario de Avaliação das Questões**: versão em PDF do questionário aplicado aos avaliadores.
- **Link do formulário (Google Forms):**
  https://forms.gle/cGkikkyhB8L1DYN4A

## Aplicação

O questionário foi respondido por três professores licenciados em Matemática.
Os avaliadores analisaram as questões propostas utilizando uma escala Likert de 1 a 5 para os critérios definidos na pesquisa.
